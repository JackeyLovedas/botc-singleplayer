# Product Scope

## 固定范围

```text
剧本：梦殒春宵
普通玩家：12
真人：1
AI：11
自动说书人：1
旅行者：不启用
传奇角色：不启用
实验角色：不启用
平台：Windows 本地应用
```

## 第一版必须支持

- 官方《梦殒春宵》角色池
- 自定义角色池
- 12 人合法配板
- 随机配板
- 固定随机种子
- 锁定角色
- 排除角色
- 完全指定实际配板
- 指定真人角色
- 部分指定 AI 角色
- 随机分配剩余角色
- 夜间行动
- 公开讨论
- 私聊
- 提名
- 投票
- 处决
- 死亡
- 疯狂
- 角色变化
- 阵营变化
- 特殊胜利
- 存档
- 回放
- 真相复盘

## 第一版暂不支持

- 旅行者
- 传奇角色
- 实验角色
- 在线多人联机
- 语音识别
- 直播模式
- 跨剧本全部角色
- 移动端
- 用户自定义角色能力脚本

不得擅自扩大第一版范围。
