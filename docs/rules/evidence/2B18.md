## Rule claims

- `officialAbilitySummary`：每夜得知自黎明以来，多少名玩家的能力因另一角色能力而异常；不检测自身能力失败，也不统计醉酒/中毒状态本身。
- `chineseAbilitySummary`：从上个黎明到被唤醒；同一玩家即使多次异常或更换角色也至多放置一枚异常标记。
- `firstNightOrder`：官方表索引76，Seamstress之后、dawn之前。
- `otherNightOrder`：索引95，Seamstress之后、dawn之前。
- `sinceDawnWindowDefinition`：通常为上个黎明至本次数学家求值时。
- `firstNightWindowStart`：**未解决**；首夜表从dusk开始，没有“上个黎明”，来源未授权用setup完成或dusk替代。
- `firstNightWindowEnd`：本次数学家求值时；不包含其随后交付的自身信息。
- `countPlayersNotIncidents`：按玩家，不按异常事件。
- `samePlayerMultipleAbnormalities`：同一玩家计1。
- `ownAbilityExclusion`：单一数学家排除该玩家自身能力；两个不同玩家拥有数学家能力时如何互相检测未规定。
- `abnormalAbilityDefinition`：实际结果因另一角色能力偏离本应结果，包括本应真却假、本应假却真、或本应生效却未生效。
- `normalOutcomeDefinition`：实际结果仍符合正常能力行为。
- `falseInformationDefinition`：陈述、手势或角色标记表达的命题不真实。
- `failedEffectDefinition`：另一角色能力导致本应发生的效果未发生。
- `registrationAbnormalityDefinition`：registration改变结果可计；官方明确以Recluse被Chef当作邪恶为例。
- `sourceDrunkRule/sourcePoisonRule`：玩家无能力但会被模拟；信息可真可假，通常应有害。
- `correctInfoWhileImpairedRule`：正确结果不计异常。
- `vortoxRule`：每名获得假信息的镇民按玩家计异常。
- `vortoxPlusImpairmentRule`：醉酒/中毒也必须获得假信息。
- `vortoxSelfInformationRule`：当前数学家的假数字不回计自身；另一数学家是否统计它未规定。
- `philosopherDuplicateRule`：哲学家获得能力，在场原角色玩家醉酒；夜间能力应在该角色本来行动时行动。
- `snakeCharmerRule`：有效非恶魔无交换正常；受损选择非恶魔无交换仍正常；受损选择恶魔却不交换为异常；正常交换不异常。
- `witchRule/pendingEffectRule`：选择和pending curse尚非异常；目标提名后本应死亡却因女巫受损未死时才计。
- `cerenovusRule`：有效疯狂效果正常；受损导致疯狂效果未施加时异常，非等待可选处决。
- `clockmakerRule`：交付等于正确距离正常；因醉酒/中毒/Vortox交付错误距离异常。
- `dreamerRule`：正常结果包含目标真实角色及一个相反阵营角色；Vortox下真实角色必须缺席；SOURCE_IMPAIRED不自动等于异常。
- `seamstressRule`：答案等于真实阵营关系正常；受损仍正确不计；错误或Vortox反值计异常。
- `evilTwinRule`：正确配对互认正常；未实现胜负/死亡不可伪造异常。
- `systemInformationRule`：MINION_INFO/DEMON_INFO不是玩家角色能力结果，不计。
- `unresolvedOutcomeRule`：规则没有软件UNRESOLVED值；产品可用它保守记录证据不足并fail closed，不得视为NORMAL或0。
- `candidateNumberDomain`：固定12人时真实计数为`0..11`；醉酒/中毒/Vortox可交付的完整假数字域未被来源限定，无法确认是`0..11`、包含12或更宽。
- `storytellerDiscretion`：有效且无Vortox必须真；受损可真可假；Vortox必须假；“最小假值”仅能是产品策略。
- `baseMathematicianRule`：适用上述规则。
- `philosopherGainedMathematicianRule`：应在数学家行动位行动；仓库当前紧随哲学家的插入顺序与官方文字冲突。
- `duplicateMathematicianRule`：**未解决**；来源未规定两名持有者的相对顺序及是否检测另一人的数学家异常。
- `explicitOutOfScope`：通用昼夜/reset、后续夜晚、提名/处决/死亡、Witch触发、Cerenovus处决、通用中毒、No Dashii、Sweetheart、registration、Summoner jinx、Traveller、Pit-Hag、Barber、UI、AI、持久化、phase transition、2B19。
