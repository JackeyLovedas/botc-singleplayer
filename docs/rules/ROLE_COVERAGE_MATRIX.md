# Sects & Violets Role Coverage Matrix

This baseline records accepted repository implementation facts only. Its statuses are implementation coverage, not the Phase One rule-research verification labels. It is not external rule evidence and does not authorize a new slice.

Cell meanings:

- `PARTIAL: ...` — an accepted implementation covers the named path but not the full dimension.
- `SKELETON: ...` — structural metadata or a marker exists without complete behavior.
- `NOT_IMPLEMENTED` — accepted status documentation explicitly leaves the behavior out.
- `UNEVALUATED` — no accepted implementation fact was established for this dimension; this makes no claim about the actual BOTC rule.

Role-catalog inclusion alone is not behavioral implementation. No role is `COMPLETE`. Dreamer remains `PARTIAL` because Vortox forced-false behavior, Storyteller false-role selection, Traveller target restriction, and complete drunk/poison information evaluation are not implemented. Seamstress is overall `PARTIAL`: the reviewed fixed-roster first-night base and Philosopher-granted choice/spend/information pipeline, represented impairment/Vortox behavior, and source-only historical projection are partial; other-night recurrence, life state, Travellers, registration, Barista, No Dashii poison derivation, and general role/effect lifecycle remain unsupported.

Repository evidence used for this baseline: accepted implementation status files `docs/implementation/phase-3-slice-2b1-status.md`, `phase-3-slice-2b4-status.md`, and `phase-3-slice-2b6-status.md` through `phase-3-slice-2b14-status.md`, plus the accepted implementation `phase-3-slice-2b15-status.md`, cross-checked against current production modules. Slice 2B15 uses the reviewed evidence in `docs/rules/evidence/2B15.md` and the independently approved v3/v3.1 design; no additional external research was performed in this matrix update.
